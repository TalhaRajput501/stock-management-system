# 🏪 Stock Management System

A **full-stack inventory and stock management platform** built to help small businesses efficiently track their products, sales, and stock levels — all in one clean dashboard.

This system was designed with **real-world business needs** in mind: managing multiple users, preventing stock mismatches, and making inventory insights accessible even for non-technical users.

---

## 🚀 What This Project Offers

### 🧾 Smart Inventory Control
Add, update, or remove products while automatically tracking quantities and stock levels.

### 👥 Role-Based Access
Two user roles — **Admin** and **Employee** — to maintain control and security.  
- **Admin:** full CRUD access + user monitoring  
- **Employee:** add, update, and view data only  

### 📈 Dashboard
See key insights at a glance — including stock levels, product categories, and out-of-stock alerts — visualized through charts and stats.

### 🧑‍🤝‍🧑 Multi-User Support
Designed for shops or medical stores with multiple staff working on the same database.

### 🎨 Clean & Responsive UI
Built with **React** and **Tailwind CSS** for a fast, modern interface that works seamlessly on both desktop and tablet screens.

---

## 💡 Problem It Solves

Managing product inventory manually or through Excel becomes messy and unreliable — especially when multiple employees are involved.  
Small businesses often struggle with:

- Duplicate entries  
- Missing stock records  
- No central system to monitor who changed what  
- Difficulty tracking sales and out-of-stock items  

The **Stock Management System** solves these by providing a **centralized, role-based web application** where:

- Every update is consistent and tracked  
- Stock levels adjust automatically  
- Admins have a bird’s-eye view of everything happening in real time

## 🧠 How I Built It

This project was developed as a **practical MERN stack application** to strengthen my full-stack skills while solving a real-world problem.

### ⚙️ Tech Highlights
- **React (Vite + Tailwind)** — for building a responsive, component-based UI  
- **Node.js + Express** — REST API backend with clean modular routes  
- **MongoDB (Mongoose)** — for efficient product and user data storage  
- **JWT Authentication + bcrypt** — for secure login and role management  
- **Redux Toolkit** — for centralized state management across the app  
- **Recharts** — for dynamic dashboard visualizations
